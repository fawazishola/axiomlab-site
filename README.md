# axiomlab-site

Minimal coming-soon page for `axiomlab.ca`.

## Files

- `index.html` — landing page
- `vercel.json` — lightweight Vercel config

## Launch

Deploy this folder as a static Vercel project.

Then connect:

- `axiomlab.ca`
- `www.axiomlab.ca`

Recommended DNS:

- `A` record: `@` -> `76.76.21.21`
- `CNAME`: `www` -> `cname.vercel-dns-0.com`
